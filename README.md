# markdown-example

### markdown是一种轻量级标记语言，具有纯文本格式语法。它由John Gruber 与Aaron Swartz在2004年合作创建。其目标是让人们“使用易于阅读且易于编写的纯文本格式进行编写，可选将其转换为结构上有效的XHTML或HTML”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性 [文档](https://www.markdownguide.org/basic-syntax/)
